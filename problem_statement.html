Welcome to Git: Stashing changes - Basic (Python).

This question is used to evaluate your skills on stashing the changes in Git.

For this, you will be evaluated based on the following operations. The remote repository has been preconfigured into your local git repository.

  1. Your local git repository will be linkedlist_python.
  2. Push the master branch of local git repository to the remote repository.
  3. Create a new branch with the name new_branch.
  4. Make a change to the Python program file (do not remove it), make a commit and push the branch.
  5. Make a few more changes to the Python program file and stash those changes.

Passing marks: 100%

Note 1: To get familiarized with the Hacker Rank environment, please watch the tour video.

Note 2: The branch names must not be misspelled.

Note 3: The IDE's Git will not work in the local repository. Use Terminal for it.
